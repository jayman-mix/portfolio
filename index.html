Budget Calculator with Conditional Image Display

I developed a budget calculator that displays one of two images based on specific field results. If the final result is between 0.1 and 2, a negative image appears. If the result is 3 or higher, a positive image is shown. The image displayed is determined by the value of the last result field.

Used the free version of the Cost Calculator Builder plugin to implement the functionality. Custom jQuery was added to handle the conditional logic, but compatibility issues caused the code to load before the plugin, preventing it from working correctly. I resolved this by using setInterval to delay the custom code, ensuring the plugin loads first and the feature functions as required by the client.

WordPress Elementor Pro jQuery
Floor Plan Display with Accordion and Tab Navigation

Developed an interactive floor plan viewer to showcase various floor plans for different room variations. I used a tab element to organize room variations and integrated an accordion within each tab to display available floor plans.

To enhance user experience, I added custom navigation buttons, allowing users to cycle through floor plans without manually clicking each tab. This streamlines navigation and improves accessibility.

The project was built using Elementor Pro, leveraging its Flex container and accordion box features. I implemented custom jQuery to add "Next" and "Previous" buttons, enabling seamless browsing between floor plans as per the client’s requirements.
